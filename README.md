## 3. 통신 방법
### 3.1. pc -> boat
-----
#### 3.1.1. 속도 제어
  - 0: zeroSpeed    00000
  - 1: 1(25%)       00100
  - 2: 2(50%)       01000
  - 3: 3(75%)       01100
  - 4: FullSpeed    10000
  - 5가지 -> 3비트

#### 3.1.2. 방향 제어
  - 0: 왼           00000
  - 1: 우           00001
  - 2: 중           00010
  - 3가지 -> 2비트

#### 3.1.3. 신호 전달 방식
신호 빈도: 10Hz  
속도 신호와 방향 신호를 OR 비트연산하여 총 5비트로 전송

### 3.2. boat -> pc
-----
#### 3.2.1. 배터리 잔량
  - 6비트 -> 64가지
  - 최소 표현단위 1.5625

## 4. 조작법
### 4.1. PC
#### 4.1.1. 속도
  - z:      속도   0%
  - x:      속도  25%
  - c:      속도  50%
  - v:      속도  75%
  - space:  속도 100%
  - w: 속도 25% 증가
  - s: 속도 25% 감소

#### 4.1.2. 방향
  - 화살표 왼쪽: 좌현으로 선회
  - 화살표 오른쪽: 우현으로 선회

#### 4.1.3. 부가 기능
  - ctrl+q: 종료
