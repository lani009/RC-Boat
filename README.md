## 3. 통신 방법
### 3.1 pc -> boat
#### 1. 속도 제어
  - 0: zeroSpeed    00000
  - 1: 1(25%)       00100
  - 2: 2(50%)       01000
  - 3: 3(75%)       01100
  - 4: FullSpeed    10000
  - 5가지 -> 3비트

#### 2. 방향 제어
  - 0: 왼           00000
  - 1: 우           00001
  - 2: 중           00010
  - 3가지 -> 2비트

#### 3. 신호 전달 방식
신호 빈도: 10Hz
속도 신호와 방향 신호를 OR 비트연산하여 총 5비트로 전송
